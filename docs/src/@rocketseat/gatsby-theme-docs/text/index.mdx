# IAM Controle de Acesso Centralizado

A ferramenta de código aberto (OpenSource) IAM da Thalamus é um sistema SSO (Single Sign-On) que gerencia o acesso centralizado a diversas aplicações.

O IAM é o sistema de autênticação e autorização utilizado em todas as aplicações THALAMUS, disponível para que comunidade.
